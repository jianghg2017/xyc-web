## 需求描述

开发一个公司官网 Web 系统，需要包含：
- 首页（公司介绍、核心业务、新闻动态）
- 关于我们（公司简介、团队介绍、发展历程）
- 产品中心（产品展示、技术优势）
- 解决方案（行业解决方案、客户案例）
- 新闻资讯（公司新闻、行业资讯）
- 联系我们（联系方式、在线留言）

## 技术栈建议

- 前端：React + TypeScript + TailwindCSS
- 后端：Node.js + Express
- 数据库：MySQL/PostgreSQL
- 部署：Docker + Nginx
