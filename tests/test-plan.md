## 测试用例示例

### TC-001: 首页加载
**步骤**:
1. 打开浏览器
2. 访问 http://localhost:5173
3. 检查页面元素

**预期结果**:
- 页面正常加载
- Hero Banner 显示
- 导航栏正常
- 产品/新闻数据从 API 加载

### TC-002: 联系表单提交
**步骤**:
1. 访问联系我们页面
2. 填写表单（姓名、邮箱、内容）
3. 提交表单

**预期结果**:
- 表单验证通过
- 提交成功提示
- 数据保存到数据库

### TC-003: 管理员登录
**步骤**:
1. 访问 /admin/login
2. 输入正确的用户名和密码
3. 点击登录

**预期结果**:
- 登录成功
- 跳转到 Dashboard
- JWT 令牌存储
- 导航栏显示管理员选项
