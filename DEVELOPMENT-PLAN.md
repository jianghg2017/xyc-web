## 🔧 技术规范

### 图片管理
```
backend/public/images/
├── products/          # 产品图片
│   ├── product-1.jpg
│   └── ...
├── news/             # 新闻配图
│   ├── news-1.jpg
│   └── ...
├── banners/          # 轮播图
│   └── banner-1.jpg
├── team/             # 团队照片
│   └── member-1.jpg
└── logo/             # Logo 和图标
    ├── logo.png
    └── favicon.ico
```

### API 规范
```javascript
// 统一响应格式
{
  "success": true,
  "data": { ... },
  "message": "操作成功",
  "timestamp": "2026-04-14T10:54:00.000Z"
}

// 分页格式
{
  "success": true,
  "data": {
    "list": [...],
    "pagination": {
      "page": 1,
      "pageSize": 10,
      "total": 100,
      "totalPages": 10
    }
  }
}
```

### 前端 API 调用
```typescript
// 统一使用 axios 封装
const apiClient = axios.create({
  baseURL: 'http://localhost:3000/api',
  timeout: 10000,
  headers: { 'Content-Type': 'application/json' }
});

// 使用示例
const { data } = await apiClient.get('/news', {
  params: { page: 1, pageSize: 10 }
});
```
