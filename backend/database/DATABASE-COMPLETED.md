## 🔧 技术特性

### 1. 索引优化
每张表都创建了适当的索引：
- 主键索引
- 外键索引
- 查询字段索引
- 全文索引 (新闻、产品搜索)

### 2. JSON 字段支持
使用 MySQL 5.7+ 的 JSON 类型存储复杂数据：
- 产品功能特性
- 产品图库
- 新闻标签
- 网站设置

### 3. 状态管理
所有表都包含状态字段：
- `status` - 启用/禁用
- `sort_order` - 排序权重
- `created_at` / `updated_at` - 时间戳

### 4. 外键约束
- `contacts.replied_by` → `admins.id`
- 支持级联删除

